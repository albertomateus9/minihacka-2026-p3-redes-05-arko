# Arko

Site estático do projeto **Arko**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores (Integrado). Esta versão funciona como um exemplo genérico e temático do produto, com dados fictícios e uma prévia interativa.

## Ideia

Um site para visualizar disponibilidade e solicitar salas, quadras, auditórios e laboratórios, com etapa de autorização pela coordenação.

## Exemplos temáticos

- **Calendário da quadra:** Professor consulta sexta-feira à tarde e vê horários livres antes de pedir reserva.
- **Pedido de laboratório:** A solicitação registra turma, horário, sala e justificativa pedagógica.
- **Aprovação da coordenação:** Pedidos pendentes ficam visíveis para aprovação ou ajuste.
