# Arko

Site estático do projeto **Arko**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores Integrado.

## Versão publicada

A página publicada parte da entrega da equipe e apresenta a proposta do Arko para organizar pedidos e autorizações de espaços escolares. A versão final preserva a identidade do projeto, a logo da EETEPA, os benefícios descritos pela equipe e uma agenda ilustrativa de reservas.
